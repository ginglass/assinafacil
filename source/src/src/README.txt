Quick install guide.

For simple usage:
    1 - Download that jar file
    2 - Download lib-deps.zip
    3 - Extract zip content to the same directory that you put AssinFacil.jar file
    4 - Insert your device (token, smartcard,...)
    5 - Double click at jar
    6 - Choose a file
    7 - Select you certificate (ALIAS)
    8 - Click on "Assinar" button
    9 - And you got choosedfile.p7s ... you brand new signed file!

If the selected file is already signed (PKSC7 signed file) you'll include a new signer on that file...
